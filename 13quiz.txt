Quiz 13: Mastering Type Guards in TypeScript


Question 01:
What is the primary purpose of a type guard in TypeScript?

-To narrow down the type of a variable within a specific scope


Question 02:
How does the typeof operator function as a type guard?

-It determines the runtime type of a variable and refines its type within a conditional block


Question 03:
What is the advantage of using the "in" operator in TypeScript type guards?

-It checks if a property exists on an object, allowing TypeScript to narrow the type accordingly


Question 04:
What is the role of a discriminated union in TypeScript?

-To allow a union of types that are differentiated by a unique literal property


Question 05:
Why is the never type useful in exhaustive checks for discriminated unions?

-It ensures that all possible cases in a union type are handled, improving type safety
